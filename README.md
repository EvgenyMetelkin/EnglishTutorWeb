# English Tutor — Репетитор английского

Одностраничное веб-приложение (SPA) для практики английского языка в формате чата с ИИ-репетитором. Поддерживает облачные модели (DeepSeek, OpenAI, Anthropic) и локальные модели через Ollama.

## Возможности

- **Облачные модели:** DeepSeek (по умолчанию), OpenAI, Anthropic — потоковые ответы через серверный прокси
- **Локальные модели:** любая модель Ollama, прямое подключение из браузера
- **Авторизация:** парольная защита через JWT, API-ключи хранятся только на сервере
- **Голосовой ввод:** распознавание речи через Web Speech API (Chrome/Edge/Safari)
- **5 стилей репетитора:** естественный диалог, исправление, объяснение, совет, личный репетитор с планом урока
- **Выбор языка общения:** русский / English
- **Светлая / тёмная тема:** автоопределение + ручной переключатель
- **Сохранение настроек и истории чата** в `localStorage`
- **Прерывание генерации:** кнопка «Стоп» или новый ввод обрывает текущий ответ
- **Потоковый вывод:** ответы выводятся посимвольно

## Требования

- **Node.js 18+** (встроенный `fetch`)
- Любой современный браузер (Chrome/Edge для голосового ввода)
- **Ollama** — опционально, только для локальных моделей

## Локальный запуск

### 1. Клонируйте проект

```bash
git clone https://github.com/EvgenyMetelkin/Web.git
cd Web
git checkout feat/cloud-models-auth-speech
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Создайте файл `.env`

Скопируйте шаблон и заполните переменные:

```bash
cp .env.example .env
nano .env
```

**Обязательные переменные:**

```bash
APP_PASSWORD_HASH=<bcrypt-хеш пароля для входа>
JWT_SECRET=<любая случайная строка>
```

**API-ключи облачных провайдеров** (укажите те, которые планируете использовать):

```bash
DEEPSEEK_KEY=sk-...           # https://platform.deepseek.com/api_keys
OPENAI_KEY=sk-...             # https://platform.openai.com/api-keys
ANTHROPIC_KEY=sk-ant-...      # https://console.anthropic.com/
```

**Порт** (опционально, по умолчанию 3000):

```bash
PORT=3000
```

### 3a. Сгенерируйте хеш пароля

```bash
npm run hash "ваш-пароль"
```

Скопируйте вывод и вставьте в `.env` как значение `APP_PASSWORD_HASH`.

### 4. Запустите сервер

```bash
npm start
```

Вывод: `English Tutor server: http://localhost:3000`

### 5. Откройте приложение

Перейдите в браузере на **http://localhost:3000**

Введите пароль из шага 3a. После входа:
- Выберите провайдера (DeepSeek / OpenAI / Anthropic / Ollama)
- Выберите модель
- Начните чат

### 6. (Опционально) Подключите Ollama

Если хотите использовать локальные модели:

```bash
# Установка: https://ollama.com/download
ollama pull llama3.2:3b
ollama serve
```

Проверьте CORS, если сервер запущен не на localhost:

```bash
OLLAMA_ORIGINS="*" ollama serve
```

## Стили репетитора

| Стиль | Поведение |
|-------|-----------|
| Естественный диалог | Предлагает более живой вариант, если фраза звучит неестественно |
| Исправление предложения | Только исправленный вариант, без комментариев |
| Объяснение ошибки | Кратко объясняет, что неверно + исправление |
| Краткий совет | Короткое правило или мнемоника |
| Личный репетитор | Проводит урок по плану: исправляет, объясняет, задаёт вопросы |

## Возможные проблемы

| Симптом | Решение |
|---------|---------|
| «Не удалось получить список моделей» | Ollama не запущен или не задан `OLLAMA_ORIGINS` |
| Ошибка «Session expired» | JWT истёк — перезайдите (выйдите и войдите снова) |
| Ошибка «Cloud API error» при отправке сообщения | Проверьте API-ключ в `.env` и баланс на платформе |
| Голосовой ввод не работает | Требуется Chrome/Edge + HTTPS (или localhost) |
| Микрофон не появляется | Откройте в Chrome/Edge — Firefox не поддерживает Web Speech API |
| Пустая страница | Убедитесь, что открываете через `http://`, не `file://` |
| Ответы медленные | Для облачных моделей — задержка сети; для Ollama — выберите модель поменьше |
